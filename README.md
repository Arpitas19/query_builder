# Query Builder

Interactive query builder app built with React, TypeScript, and Vite.

## Repository

This project is intended to be hosted as a **public GitHub repository** with this README included.

## Setup

### Prerequisites

- Node.js 18+
- npm 9+

### Install dependencies

```bash
npm install
```

### Run in development

```bash
npm run dev
```

After running the command, open the local URL shown in the terminal (typically `http://localhost:5173`).

## Available Scripts

- `npm run dev` - Start Vite dev server
- `npm run build` - Type-check and build for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint

## Tech Stack

- React
- TypeScript
- Vite
- ESLint
